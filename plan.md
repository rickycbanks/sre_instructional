# SRE Interactive Learning Path - Project Plan

This document outlines the roadmap for building the SRE Interactive Learning Path, a platform designed to teach Site Reliability Engineering using Google SRE as a baseline, integrated with multi-cloud (AWS, Azure, GCP) and DevOps practices.

## 🛠 Tech Stack
- **Framework:** Next.js (TypeScript)
- **Styling:** Tailwind CSS
- **Content:** MDX (Markdown with React Components)
- **Deployment:** Vercel (recommended)

## 📚 Curriculum Roadmap

### Module 1: Foundations & Mindset
- Introduction to SRE vs. DevOps.
- The concept of "Toil" and how to measure it.
- SRE Principles (The 7 Pillars).

### Module 2: The Math of Reliability (SLO/SLI)
- Defining SLIs and setting SLOs.
- Error Budgets: The "Gas Tank" of innovation.
- **Interactive:** SLO Calculator.

### Module 3: Observability (The 4 Golden Signals)
- Latency, Traffic, Errors, and Saturation.
- Implementing monitoring with Prometheus and Grafana.
- Distributed Tracing and Logging (OpenTelemetry).

### Module 4: Multi-Cloud & Infrastructure
- Comparison of AWS, Azure, and Google Cloud services.
- Infrastructure as Code (Terraform) for multi-cloud.
- Linux Administration and Performance Tuning.

### Module 5: Incident Management & Culture
- The Incident Lifecycle (Detection to Resolution).
- Blameless Post-mortems.
- Reliability Engineering: Chaos Engineering and Load Testing.
